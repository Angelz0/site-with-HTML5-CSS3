Post #2: Como compartilhar seu projeto no GitHub (Windows)

Primeiro: Criar um novo repositório no github.com, colocar o nome do projeto e a descrição.

No windows, com o Git Bash já instalado (você pode baixar aqui: https://gitforwindows.org), faça o seguinte passo-a-passo (é simples e sempre dá certo):
  1. Clicar com o botão direito na pasta a ser adicionada e depois em “Git Bash Here”
  2. git init (inicializar a pasta a ser adicionada)
  3. git status (lista tudo o que tem dentro da pasta)
  4. git add *
  5. git status (lista tudo o que será enviado para o GitHub)
  6. git commit -m “project code”
  7. git remote add origin + url do repositório que você criou
  8. git push -u origin master
  9. Digitar o seu usuário e a sua senha
  10. O seu projeto já estará no repositório criado.

Post #1: Compilar código Java pelo prompt

Depois de escrever seu código Java no bloco de notas e salvar com o nome da classe + extensão .java, faça o seguinte passo-a-passo:

  1. Abra o prompt (cmd) do Windows;
  2. Digite cd + caminho do diretório onde o arquivo se encontra (se estiver na área de trabalho, usar cd + Desktop);
  3. Digitar o comando javac + nome do seu arquivo.java (por exemplo, para um arquivo que se chama "MeuPrograma", digitar "javac MeuPrograma.java");
  4. O arquivo já estará compilado e você poderá executá-lo digitando java + nome do arquivo (por exemplo, java MeuPrograma);

Lembrando que para compilar e executar pelo prompt é necessário já ter a JRE instalado e que escrever, compilar e executar um código é muito mais fácil usando uma IDE, como o Eclipse ou o NetBeans.
